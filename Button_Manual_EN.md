# Button Manual (EN)

## Top Bar
- Save: stores API/root locally.
- Health Check: verifies backend `/api/health`.

## Output Page
- Generate Submission Blocks: output markdown submission blocks.
- Generate HC Check: output HC validation report.
- Document Field Health Check: detects missing fields and special cases.
- Analytics Dashboard: outputs statistical report + risk CSV + dashboard JSON.
- Batch Sequential: run selected action on multiple CSVs in order.
- Mask Upload (Auto Recover Output): mask sensitive columns and auto-recover outputs.

## Center Check Page
- Save/Load/Reset Template: maintain default parameters.
- Export/Import Template JSON: move templates across computers.
- Build Allocation (HC): create allocation CSV from headcount + params.
- Auto Fill and Run: fill defaults and execute allocation directly.
- Check Centers (Upload Status File): validate with user status file.
- Check Centers (Use Default Status File): validate with default status template.

## Ledger Page
- Update Ledger: merge prefill data into ledger output.

## Schedule Page
- Mark Schedule Configured: mark schedule step as done.

## History Page
- Query history by region / step / time range.

## Queue Page
- Enqueue, Start, Pause, Resume, Retry Failed, Export Queue Log.
