# 工作日前置预检（定时任务）说明

## 目标
每个工作日上午自动跑一次预检，提前发现缺资料、失效成本中心、台账风险。

## 包含动作
1. 目录预检（命名/缺文件/期间）
2. 成本中心有效性检查

## 一键安装
- 双击：`一键安装_工作日前置预检任务.command`
- 安装后会创建 LaunchAgent（macOS）
- 默认执行时间：工作日 09:00

## 执行脚本
- `工作日前置预检任务.sh`
- 日志输出到：`运行日志/`

## 卸载（macOS）
```bash
launchctl unload ~/Library/LaunchAgents/com.fpp.workday.precheck.plist
rm ~/Library/LaunchAgents/com.fpp.workday.precheck.plist
```
