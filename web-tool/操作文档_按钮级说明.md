# 操作文档（按钮级说明）

## 页面总览
- 页面分为：顶部连接区、左侧导航、右侧业务区、结果区、文档区。
- 国家维度严格分离：`US Workspace` 与 `CAN Workspace` 的按钮、模板、结果互不影响。

## 顶部连接区（全局）
- `当前：国家 / 步骤`（面包屑）
  - 作用：实时显示你当前所在国家和功能步骤，避免迷路。
- `保存`
  - 作用：保存当前 `Backend URL` 和 `Root Path` 到本机浏览器。
  - 结果：下次打开页面自动带入。
- `健康检查`
  - 作用：请求后端 `/api/health`。
  - 成功：显示 `ok: true` 的 JSON。
  - 失败：显示连接失败信息。
- `安全说明（绿色提示条）`
  - 含义：文件本地处理，不上传外网。

## 左侧导航
- `US Workspace` / `CAN Workspace`
  - 作用：切换国家工作区。
- `1) 文档产出`、`2) 成本中心预检`、`3) 台账回填`、`4) 定时预检`
  - 作用：切换当前国家下的效能分页。
- `5) 执行历史`
  - 作用：按国家/步骤/时间筛选历史执行记录。
- `6) 任务队列`
  - 作用：批量任务排队执行，可暂停/继续/失败重试。
- 右侧状态标签
  - `未开始`：此步骤尚未执行成功。
  - `进行中`：步骤请求已发起，等待结果。
  - `已完成` + 绿色 `✓`：步骤至少成功执行一次。
- 进度条
  - 按 4 个步骤统计完成率，例如 `50% (2/4)`。

## 1) 文档产出（当前国家）
- `生成提单文本`
  - 输入：提单预填 CSV。
  - 输出：`FPP提单文本块_<国家>_时间戳.md`
  - 完成态：步骤1点亮。
- `生成HC校验`
  - 输入：HC Analysis CSV。
  - 输出：`HC校验报告_<国家>_时间戳.md`
  - 完成态：步骤1点亮。
- `批量串行执行`
  - 输入：多选 CSV + 选择动作（批量提单文本/批量HC校验）。
  - 行为：按文件顺序串行调用，逐个产出结果。
  - 完成态：步骤1点亮。

## 2) 成本中心预检（当前国家）

### 模板按钮
- `保存默认模板`
  - 把期间/国家/主体/供应商/费用类型/金额/币种保存到本机。
- `读取默认模板`
  - 读取并回填本机已保存模板。
- `重置默认模板`
  - 清空该国家模板，恢复系统默认值。
- `导出模板JSON`
  - 导出本地模板到 JSON，供新电脑导入。
- `导入模板JSON`
  - 导入 JSON 并自动校验字段完整性，不完整会弹窗提示并拦截。

### 执行按钮
- `自动分摊生成HC`
  - 输入：人数 CSV + 参数字段。
  - 输出：`HC_Analysis_<国家>_时间戳.csv`
  - 完成态：步骤2点亮。
- `检查成本中心（上传状态表）`
  - 输入：source CSV + status CSV。
  - 输出：`成本中心有效性检查报告_<国家>_时间戳.md`
  - 完成态：步骤2点亮。
- `检查成本中心（用默认状态表）`
  - 输入：source CSV（状态表走默认路径）。
  - 输出：`成本中心有效性检查报告_<国家>_时间戳.md`
  - 完成态：步骤2点亮。

## 3) 台账回填（当前国家）
- `台账自动更新`
  - 输入：prefill CSV（必填）+ ledger CSV（可选）。
  - 输出：`台账_自动更新_<国家>_时间戳.csv`
  - 完成态：步骤3点亮。

## 4) 定时预检（当前国家）
- `已完成定时配置（打勾）`
  - 作用：手动确认该国家定时任务已配置。
  - 完成态：步骤4点亮。
- 文档链接：
  - `工作日前置预检说明`
  - `一键安装定时任务脚本`
  - `定时执行脚本`

## 结果区
- `执行摘要`（默认展开）
  - 成功：显示“已生成X个文件+下载提示”。
  - 失败：按类型给修复建议（字段错误/文件错误/服务错误）。
- `查看原始JSON`（折叠区）
  - 仅在需要排错时展开查看。
- 输出链接
  - 点击下载产物文件。

## 5) 执行历史（当前国家）
- `查询历史`
  - 输入：步骤（可选）、开始时间（可选）、结束时间（可选）。
  - 输出：历史表格（时间、国家、步骤、动作、状态、耗时）。

## 6) 任务队列（当前国家）
- `加入队列`
  - 把当前多选 CSV 按指定动作加入队列。
- `自动重试次数`
  - 默认 `3`，表示单条任务失败后自动重试最多 3 次。
- `开始执行`
  - 从队列头开始串行执行。
- `暂停`
  - 当前批次执行在下一文件前暂停。
- `继续`
  - 从暂停点继续。
- `失败重试`
  - 将失败项重新置为待执行并可再次运行。
- `取消`（队列表内单条按钮）
  - 仅可取消 `queued/failed` 项，运行中任务需先暂停等待当前任务结束。
- `导出队列记录`
  - 导出当前国家队列为 CSV（含状态、重试次数、最近执行时间、错误信息）。

## 首次引导（3步浮层）
- 首次打开自动弹出：
  1. 先健康检查
  2. 选国家和步骤
  3. 看摘要并下载产物
- 可 `跳过`，跳过后本机不再重复弹出。

## 常见问题
- 点击按钮无反应：先做 `健康检查`。
- 步骤一直“进行中”：检查后端日志或网络；失败会回到“未开始”。
- 导入模板失败：JSON 缺字段或格式错误，按弹窗修正后重试。
