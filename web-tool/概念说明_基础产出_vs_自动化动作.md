# 概念说明：文档产出 vs 自动化动作

## 文档产出是什么
- 目标：把输入文件整理成可提交、可复制、可复核的输出文档。
- 典型结果：`FPP提单文本块_*.md`、`HC校验报告_*.md`。
- 特点：不改业务台账，只做“产物生成”。

## 自动化动作是什么
- 目标：减少重复录入和人工判断，把常规步骤自动执行。
- 典型结果：`HC_Analysis_*.csv`、`成本中心有效性检查报告_*.md`、`台账_自动更新_*.csv`。
- 特点：会对输入做计算、校验、去重更新。

## 怎么理解两者关系
1. 先做“文档产出”确认数据正确；
2. 再做“自动化动作”完成预检、回填和日常运行；
3. 两者都保留文件输出，方便审计与交接。
