# 案例文档（操作者演练版）

## 目标
让操作者在 15-20 分钟内完成一次完整演练：上传 -> 生成 -> 下载 -> 校对。

## 演练前准备
- 启动页面：`一键重启并打开上传文档页.command`
- 页面地址：`http://127.0.0.1:18081/upload-docs.html`
- 先点 `Health Check`，确认后端可用

## US 案例（推荐先做）

### 案例A：生成提单文本
- 工作区：US Workspace
- 上传文件：`提单预填表模板.csv`
- 点击按钮：`生成提单文本 / Submission Blocks`
- 预期输出：`FPP提单文本块_US_*.md`

### 案例B：HC校验
- 工作区：US Workspace
- 上传文件：`HC_Analysis_统一模板.csv`
- 点击按钮：`生成HC校验 / HC Check`
- 预期输出：`HC校验报告_US_*.md`

### 案例C：自动分摊
- 工作区：US Workspace
- 上传文件：`痛点攻坚实施包/成本中心人数输入模板.csv`
- 参数示例：
  - period: `202606`
  - country: `US`
  - entity: `Tencent America LLC`
  - vendor: `Anthem`
  - fee_type: `Medical Insurance`
  - invoice_total: `145411.03`
  - currency: `USD`
- 点击按钮：`自动分摊 / Build Allocation`
- 预期输出：`HC_Analysis_US_*.csv`

### 案例D：成本中心检查
- 工作区：US Workspace
- source 上传：`HC_Analysis_统一模板.csv`
- status 上传：`痛点攻坚实施包/成本中心状态表模板.csv`
- 点击按钮：`成本中心检查 / Check Centers`
- 预期输出：`成本中心有效性检查报告_US_*.md`

### 案例E：台账自动更新
- 工作区：US Workspace
- prefill 上传：`提单预填表模板.csv`
- ledger 上传（可选）：`痛点攻坚实施包/台账模板.csv`
- 点击按钮：`台账自动更新 / Update Ledger`
- 预期输出：`台账_自动更新_US_*.csv`

## CAN 案例
与 US 相同流程，在 CAN Workspace 操作，输出文件自动带 `CAN` 标识。

## 验收标准
- 每一步执行后，结果区显示 `ok: true`
- 结果区出现下载链接
- 输出文件命名包含国家标识（US 或 CAN）和时间戳
