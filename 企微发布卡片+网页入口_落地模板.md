# AMER FPP + ai｜企业微信“发布卡片 + 网页入口”落地模板

适用目标：把你现在的工具做成“企微里点卡片 -> 打开网页工具”  
当前状态说明：你现有系统是“本地前后端运行版”，要做卡片推广，建议先有一个可访问URL（内网服务器优先）。

---

## 0) 你现在这套，怎么接到“卡片+网页入口”

最短路径：
1. 先把网页工具部署到可访问地址（内网URL/域名）
2. 在企微机器人或应用里发“图文卡片”
3. 卡片点击后跳转到你的工具URL

> 关键点：企微只是入口，工具本体仍在网页服务上。

## 4) 你当前系统对应的部署与接入步骤（最短路径版）

### 步骤A：部署网页与后端（内网机器）
1. 把`工作提效`目录放到一台内网可访问机器
2. 启动后端（FastAPI）与前端静态服务
3. 用内网域名/反向代理暴露入口（建议HTTPS）
4. 确认访问：
   - `https://<domain>/upload-docs.html`
   - `https://<domain>/api/health`（或反代后的后端健康地址）

### 步骤B：在企微发布卡片
1. 进入机器人/应用配置页面
2. 选择图文卡片（news）类型
3. 填“标题/摘要/封面/跳转链接”
4. 先发测试群验证

### 步骤C：试运行验收（3条）
1. 点击卡片能打开网页入口
2. 网页`Health Check`通过
3. 上传`提单预填表模板.csv`后可生成输出文件
