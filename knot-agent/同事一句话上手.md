## 主菜单数字速查

| 数字 | 环节 | 需要什么 |
|------|------|---------|
| 0 | 全流程演示 | 不用准备文件 |
| 1 | 目录预检 | 当月 USA/CAN 目录 |
| 2 | 自动分摊 | 人数 CSV + 期间/账单额 |
| 3 | HC校验 | HC Analysis CSV |
| 4 | 生成提单文本 | 提单预填表 CSV |
| 5 | 检查成本中心 | HC源文件 + 状态表 |
| 6 | 台账更新 | 预填表 + 台账 |
| 7 | 总检查 | 目录 + HC |

## 每月推荐顺序

**4 → 2 → 3 → 5 → 6**（提单文本 → 分摊 → HC校验 → 成本中心 → 台账）

## 怎么确认结果可信

每条回复会有 **`DONE: /data/workspace/.../文件路径`**，打开该文件复核即可。  
助手不会口头编造金额或成本中心。
