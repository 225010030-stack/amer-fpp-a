# SOP 分步执行卡（未来账单 · 同事版）

> 在 Knot/企微 chat 发 **「主菜单」** 可看到同款引导。  
> 发数字或关键词即可进入对应环节。

---

## 总流程（6 环节 → 4 痛点）

```text
环节1 Source  → 环节2 Process → 环节3 Submit → 环节4 Exception → 环节5 Output → 环节6 Improve
  目录预检        自动分摊+HC校验    提单文本        成本中心检查        台账更新         复盘留痕
```

---

## 【0】首次体验 / 培训

| 项目 | 内容 |
|------|------|
| **你说** | `US演示` 或 `四痛点演示` |
| **需要文件** | 无（内置样例） |
| **得到** | 4 类输出文件各 1 份 |
| **下一步** | 打开 DONE 路径复核 |

---

## 【1】环节1 Source · 目录预检

| 项目 | 内容 |
|------|------|
| **你说** | `目录预检` 或 `1` |
| **什么时候做** | 每月收齐材料后、分摊前 |
| **需要文件/条件** | USA/CAN 下当月文件夹；或指定月份如 `202606` |
| **可选参数** | `目录预检 202606` |
| **输出** | `目录预检输出/预检报告_*.md` |
| **责任人** | 操作者主责；人力配合人数口径 |
| **下一步** | 按报告修目录 → 进入环节2 |

---

## 【2】环节2 Process · 自动分摊（痛点1）

| 项目 | 内容 |
|------|------|
| **你说** | `自动分摊` 或 `痛点1` 或 `2` |
| **什么时候做** | 拿到人数 CSV + 账单总额后 |
| **需要文件** | `成本中心人数输入.csv`（列：成本中心、人数） |
| **需要参数** | 期间(202606)、国家(US/CAN)、Entity、Vendor、Fee Type、Invoice Total、Currency |
| **上传方式** | 工作区文件管理 → `上传输入/` → chat 告知路径 |
| **输出** | `上传输出/HC_Analysis_{US\|CAN}_*.csv` |
| **验收** | 分摊合计 = 账单总额 |
| **下一步** | 发 `HC校验` |

---

## 【3】HC 校验

| 项目 | 内容 |
|------|------|
| **你说** | `HC校验` 或 `3` |
| **需要文件** | 上一步 HC_Analysis CSV |
| **输出** | `HC校验输出/HC校验报告_*.md` |
| **下一步** | 发 `生成提单文本块` |

---

## 【4】环节3 Submit · 生成提单文本（痛点2）

| 项目 | 内容 |
|------|------|
| **你说** | `生成提单文本块` 或 `痛点2` 或 `4` |
| **需要文件** | `提单预填表.csv` |
| **输出** | `上传输出/FPP提单文本块_*.md` |
| **用途** | 复制字段块到 FPP 系统录入 |
| **下一步** | 发 `检查成本中心`（提交前必做） |

---

## 【5】环节4 Exception · 检查成本中心（痛点3）

| 项目 | 内容 |
|------|------|
| **你说** | `检查成本中心` 或 `痛点3` 或 `5` |
| **需要文件** | Source: HC Analysis；Status: 成本中心状态表 |
| **输出** | `上传输出/成本中心有效性检查报告_*.md` |
| **阻断** | 报告里「阻断项」未清零不得提单 |
| **下一步** | 发 `台账更新` |

---

## 【6】环节5 Output · 台账更新（痛点4）

| 项目 | 内容 |
|------|------|
| **你说** | `台账更新` 或 `痛点4` 或 `6` |
| **需要文件** | Prefill 提单预填表 + Ledger 台账 |
| **输出** | `上传输出/台账_自动更新_*.csv` |
| **验收** | 无重复行、更新时间完整 |
| **下一步** | 归档输入输出 → 本月批次完成 |

---

## 【7】总检查

| 项目 | 内容 |
|------|------|
| **你说** | `总检查` 或 `7` |
| **作用** | 目录预检 + HC校验 一次跑完 |
