# 点击选择菜单 · 部署说明（像 HR 助手）

## 三种形态对照

| 形态 | 体验 | 怎么开 | 推荐 |
|------|------|--------|------|
| **A 点击菜单网页** | 真·按钮点击，像 App | `chat-menu.html` | ⭐⭐⭐ 最接近 HR 截图 |
| **B 企微/Knot 聊天** | 发数字/关键词 | 企微机器人 / chat URL | ⭐⭐ 日常 |
| **C 上传网页** | 四按钮 + 拖文件 | `upload-docs.html` | 多 CSV 时用 |

**推荐组合：B 做入口 + A 做「点击选环节」+ C 做复杂上传**

---

## A. 点击菜单网页（新建，HR 同款）

### 1. 启动后端

工作区终端：

```bash
cd /data/workspace/amer-fpp-a
bash knot-chat/start_web.sh
curl http://127.0.0.1:18082/api/health
```

### 2. 打开页面

浏览器：

```text
http://127.0.0.1:18081/chat-menu.html
```

（内网部署后换正式域名）

### 3. Knot 发布给同事

**智能体集成 → 发布为网页 URL**：

- 若只能开 chat：把 `chat-menu.html` 链接贴在 Prompt 欢迎语里
- 或 **嵌入至 web 网页** iframe 到内部门户

### 4. 页面功能

- 左侧 FPP 头像 + 聊天气泡
- **0–7 蓝色可点按钮**（每项显示需要什么文件）
- 底部：**主菜单 | US演示 | 多文件上传 | 有帮助 | 需要人工支持**
- 点击 → 调 `/api/run` → 显示 DONE 路径（不编造）

---

## B. 企微聊天里尽量像「可点选」

企微智能机器人 **本身** 的蓝色链接（如 HR 截图）需在 **详细配置** 里配模板卡片/欢迎语；Knot 若不支持原生按钮，用下面替代：

### Prompt 要求（已写入 PROMPT.md）

每条回复末尾固定 footer：

```text
---
有帮助 | 无帮助 | 需要人工支持 | 主菜单
（请回复：主菜单 / US演示 / 2 / 4 等数字）
```

### 欢迎语配置（Knot → 企微智能机器人 → 详细配置）

若有「欢迎消息/开场白」字段，粘贴：

```text
你好！我是 AMER FPP 账单助手。
请回复「主菜单」查看可选项，或打开点击菜单：
http://你的域名/chat-menu.html
```

---

## C. 文件位置

| 文件 | 作用 |
|------|------|
| `web-tool/chat-menu.html` | 点击菜单 UI |
| `bot-gateway/config/menu.json` | 菜单项配置（可改文案） |
| `GET /api/menu` | 页面读菜单 |
| `POST /api/run` | 点击后执行脚本 |

---

## 改菜单文案

编辑 `bot-gateway/config/menu.json` → 重启 backend 或 refresh 页面。

---

## 验收

1. 打开 `chat-menu.html` → 看到 0–7 按钮
2. 点 **0 US演示** → 出现多个 DONE 路径
3. 点 **主菜单** → 回到选项列表
4. 点 **多文件上传** → 跳转 upload-docs.html
