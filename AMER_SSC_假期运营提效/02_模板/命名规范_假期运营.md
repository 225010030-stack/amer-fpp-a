# 假期运营 · 上传文件命名规范

> 对齐 FPP 命名风格：`IN_{国家}_L{环节}_{说明}_{YYYYMM}.csv`  
> `L` = Leave（假期域），与 FPP 的 `P`（Payment/FPP）区分，避免混目录。

---

## 格式

```
IN_{US|CAN}_L{环节}_{说明}_{YYYYMM}.csv
```

| 前缀 | 含义 |
|------|------|
| `TPL_` | 空模板（参考） |
| `DEMO_` | 演示样例 |
| `IN_` | 同事实际上传 |

---

## 环节对照

| 步骤 | 环节 | US 示例 | CAN 示例 |
|------|------|---------|----------|
| L1 双源输入 | ADP Time Off / Timecard 导出 | `IN_US_L1_ADP_TimeOff_202606.csv` | `IN_CAN_L1_ADP_TimeOff_202606.csv` |
| L1 双源输入 | WD Absence 导出 | `IN_US_L1_WD_Absence_202606.csv` | `IN_CAN_L1_WD_Absence_202606.csv` |
| L2 映射治理 | 员工 ID 映射（Benefits 维护） | `IN_US_L2_员工ID映射_202606.csv` | `IN_CAN_L2_员工ID映射_202606.csv` |
| L2 映射治理 | 假期类型映射 | `IN_US_L2_假期类型映射_202606.csv` | `IN_CAN_L2_假期类型映射_202606.csv` |
| L3 Prepayroll | 假期结算输入包 | `IN_US_L3_PrepayrollLeaveInput_202606.csv` | `IN_CAN_L3_PrepayrollLeaveInput_202606.csv` |

## 与 FPP 文件共存规则

- 同一 `上传输入/` 目录，**FPP 用 P1–P4，假期用 L1–L3**
- US 与 CAN **整批只在一个国家工作区操作**（与 FPP 相同）
- 输出文件名必须含 `_US_` 或 `_CAN_`
