| 文件 | 说明 |
|------|------|
| `AMER_FPP_AI工作台与组织建设说明.md` | Markdown 源文件（可继续编辑） |
| `AMER_FPP_AI工作台与组织建设说明.html` | 浏览器版（排版更好，可打印为 PDF） |

### 重新生成 PDF

```bash
cd docs
python3 build_pdf.py
```

若 PDF 生成失败，可打开 `.html` → 浏览器 **打印 → 存储为 PDF**。
